How to use this CDRA Training Module

Format of the training module

This training module is compiled using a CHM compiler. The output is a Compiled Help (.chm) file that is commonly used in Windows OS. This .chm file compiles text, photos and media in a single document. The pages, sections (and subsections) and links in this file are interactive and can be easily accessed through a Table of Contents.


Intended users

  • The intended primary users of this CDRA Training Module are the personnel from HLURB (i.e. Central and Regional Officers who facilitate CDRA and CLUP formulation and updating) and other mandated government agencies (e.g. DILG, OCD-NDRRMC, CCC, etc.).
  • LGU officials and officers (e.g. Municipal/City Planning Development Office, Municipal/City DRRM Office, etc.) can also use this training module as one of their guide references in formulating CLUP. Users from LGU, however, are strongly advised to get a proper training from mandated national agencies.
  • Other users from the public and private sectors who provide social services to the people. They include officers and personnel from local water district, electric cooperative, non-government organizations, consultants and many development partners and humanitarian organizations.


Software and hardware requirement

    • For windows user, CHM reader is already integrated in the windows platform. For MAC users, CHM readers such CHM Reader and CHMOX can be downloaded at Apple Apps. For Android users, CHM readers such as Chm Reader X can be downloaded from Playstore.
    • CHM runs on a minimum hardware requirement. No specific hardware specification is required to use this CDRA Training Module.


How to open and make a copy of the CDRA in CHM format

    • For desktop or laptop users, copy the whole "CDRA" folder provided to you into the C:\ drive. Make sure that "CDRA.chm" and "lib" folder are in the same folder.
    • The CDRA.chm can also be opened directly from the USB. Just make sure that it is in the same folder as the "lib" folder.
    • To open the CDRA.chm, simply double click the file. For MAC users, make sure that you have a CHM reader installed in your computer.


Alternative access to CDRA Training Module

    • CDRA Training Module can be accessed as html web page, pdf, ebook, etc. Please contact DHSUD-Environmental, Land Use and Urban Planning and Development Bureau (ELUPDB) for any updates.


Interactive boxes and links

    • This training module compiles all relevant materials (i.e. PowerPoints, References, etc.) to facilitate CDRA. These materials can be accessed via interactive boxes and links which can be found in almost all pages of the document. Simply click a box or link and another dialog box or windows will open containing the desired materials.

GIS Software and Exercises

This training module provides videos and dummy files (i.e. shapefiles for dummy data) for GIS exercise purposes. The purpose is to guide both facilitators and users in using GIS for mapping activities related to CDRA. This can be accessed through:


    • interactive boxes above in this CHM compiler.
    • CDRA Training Module folder to be provided by the facilitator of CDRA (i.e. HLURB facilitators or any users of this CHM compiler). If the folder is copied in the C:\ drive, the GIS exercises can be located in the C:\CDRA\lib\GIS_Exer.
    • the GIS software used for this training module is QGIS 3.x version.  Offline installers of QGIS are provided in this module and can be accessed in the installers folder. LGUs are encouraged to use open-source software for many benefits it offers. However, LGUs may also use commercial GIS software such as ARCGIS and Manifold based on their preference.
